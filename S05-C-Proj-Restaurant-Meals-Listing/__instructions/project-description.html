Project Description

Restaurant Meal Listing

Use a database table to store and display a set of restaurant meals.

  • Create Database table called Meals - with the following columns; id (primary key, auto increment), imageURL, Name and Description of Meal
  • Add a record for each meal (at least 10)
  • Using PHP, connect to your database, query and display the restaurant meals.
  • Each Meal must display an image, name and description.
  • The appearance of your solution must be addressed with CSS.
