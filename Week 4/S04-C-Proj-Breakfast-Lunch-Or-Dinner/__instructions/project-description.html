Project Description

Breakfast, Lunch Or Dinner?

Create a single page that both collects the user's input and processes the results to conditionally show a restaurant menu. The 3 choices will be breakfast, lunch and dinner. When the user chooses breakfast, lunch or dinner it will display the meals for that category.

  • Use one page to both collect the users input (form) and to process it
  • Use the POST method.
  • Create initialized variables
  • Use conditions to determine which Title and Image should be displayed
  • Use Sticky Forms to make sure the select drop down checks the chosen weather (e.g. rain, cloudy, clear)
  • At least 3 meals per category
  • Each meal should have an image, title, and brief description
  • Style everything to organize the results.
