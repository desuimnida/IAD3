Project Description

Mad Libs

Create a form that collects user input and populates a short story of your choosing.

  • The form and the results must be separate files
  • Form must collect at least 10 inputs
  • Using CSS, The populated results should distinguish themselves from the surrounding unpopulated text (e.g. add background colors)
  • Short Story must be approximately 2-3 paragraphs, can be found or made up.
