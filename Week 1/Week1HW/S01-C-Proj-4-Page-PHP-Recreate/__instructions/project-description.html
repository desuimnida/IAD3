Project Description

4-Page PHP Recreate

Using the resources and goal files provided - recreate the 4 page website and use PHP to control shared assets and dynamic variables.

  • Pay special attention to spatial relationships, colors and placement of elements
  • Note the float is a different sequence than the lesson
  • Portfolio Page is the only page that does not have article and aside floated
  • Nav Links must have default, hover, and chosen states
  • No need for any JS or JQuery
  • All styles should be saved to an shared, external CSS stylesheet
  • All web pages must PHP and have the .php extension
  • Must use globals.php with at least one sitewwide shared variable (e.g. Site Title)
  • Each page must have at least 2 page-specific variables (e.g. page title, pageID)
